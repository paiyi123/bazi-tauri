# 八字排盤 App

這是一個以 `Tauri + Vue 3 + TypeScript + Rust` 開發的本機八字排盤應用程式，目標是把原本偏 Web/API 的排盤流程改成可在本機執行的桌面與行動端 App。

目前專案以 Rust 為核心計算引擎，前端提供輸入、排盤、分析與列印預覽介面，並朝 Windows 與 Android 版本共用同一套核心邏輯的方向發展。

## 這個 App 在做什麼

使用者可以輸入出生年月日時間，或直接輸入四柱，系統會在本機完成八字排盤，並顯示：

- 命盤摘要
- 四柱、藏干、十神
- 大運、流年、流月
- 起運與交脫大運資訊
- 合沖刑破等互動分析
- 用神與量化五行力分析
- 列印預覽與命盤列印

## 主要功能

### 1. 出生資料排盤

- 支援公曆與農曆輸入
- 支援西元與民國年
- 支援時分輸入與時辰輸入
- 支援晚子時流派與起運流派切換
- 由 Rust 本機引擎直接計算，不依賴外部 API

### 2. 直接輸入四柱

- 適合已知四柱的命例、書籍案例與測試資料
- 可直接輸入年、月、日、時四柱
- 支援推回可能的西元年份候選
- 保留後續四柱分析、順逆排運與量化資訊

### 3. 命盤與運勢檢視

- 命盤摘要：曆別、八字、日主、格局、胎元、命宮、身宮、起運資訊
- 四柱面板：四柱本體、藏干、十神、旬空、納音、五行等資料
- 大運面板：大運、流年、流月與目前焦點柱位預覽
- 關係面板：原局與大運加入後的合、沖、刑、破互動

### 4. 分析功能

- 用神分析
- 從格與不從分析
- 殺印相生分析
- 量化五行力
- 大運量化與流年量化結果

### 5. 列印功能

- 提供列印預覽
- 依傳統命盤版型輸出
- 預覽與實際列印共用同一套模板邏輯

## 技術架構

- 前端：Vue 3、TypeScript、Element Plus、Vite
- 本機容器：Tauri 2
- 核心計算：Rust
- 日期曆法核心：`tyme4rs`
- 目標平台：Windows、Android

## 開發指令

在專案根目錄執行：

```bash
npm install
```

### 前端開發

```bash
npm run dev
```

### Tauri 桌面開發

```bash
npm run tauri:dev
```

### 前端建置

```bash
npm run build
```

### Tauri 桌面建置

```bash
npm run tauri:build
```

### Android

```bash
npm run tauri:android:init
npm run tauri:android:dev
npm run tauri:android:build
```

也可以直接產出：

```bash
npm run tauri:android:apk
npm run tauri:android:aab
```

## 專案重點

- 本機計算：排盤與分析邏輯盡量留在 Rust 端，減少對外部服務依賴
- 多端共用：同一套核心邏輯同時服務桌面與 Android
- 傳統版型：列印輸出對齊傳統八字命盤樣式
- 漸進重寫：參考 `bazi-sources` 的流程與功能，逐步收斂到本機 App 版本
