# 第六篇《從與不從》筆記（152-164）

## 範圍
- 下冊第六篇 `152-164` 頁
- 主題：如何用量化總分判斷命局屬於 `從強`、`從弱`，還是 `不從`

## 章節核心結論
- 書中把「從格」收斂為兩大類：
  - `從強`
  - `從弱`
- `從強` 的常見子類：
  - `專旺印比`
- `從弱` 的常見子類：
  - `從兒格`
  - `從財格`
  - `從官殺格`
- 日元五行本質不改變；所謂「從」，是命局總能量已極端偏向某一側。

## 量化界線
- 書中以大量案例比較後，提出：
  - `總分 >= +210`：可視作 `從強`
  - `總分 <= -210`：可視作 `從弱`
  - `-210 < 總分 < +210`：視作 `不從`
- 書中說明此界線與全局總位置分約 `400` 有關，因此正負 `210` 可視作「極端偏向」的分界。

## 子類型判斷
- 若已跨過 `+210`
  - 再看正分主要集中在哪一類十神
  - 若印、比劫顯著佔優，可判為 `專旺印比`
  - 否則先統稱 `從強`
- 若已低於 `-210`
  - 再看負分主要集中在哪一類十神
  - `食傷` 主導：`從兒格`
  - `財` 主導：`從財格`
  - `官殺` 主導：`從官殺格`
- 書中還提到判別難度：
  - `從兒` 最容易
  - `從財` 次之
  - `從官殺` 最難

## 工程化規則

### 第一層：先看總分
- 以目前量化模型的 `totalScore` 先作第一刀判斷。
- 規則：
  - `>= +210`：進入 `從強` 分流
  - `<= -210`：進入 `從弱` 分流
  - 其餘：`不從`

### 第二層：再看主導十神家族
- 把天干分數與地支各列 `finalContribution` 依十神家族彙總：
  - 比劫
  - 印
  - 食傷
  - 財
  - 官殺
- 正分看 `從強`
- 負分看 `從弱`

### 第三層：輸出章法說明
- `pattern`
  - `從強 / 從弱 / 不從`
- `subtype`
  - `專旺印比 / 從兒格 / 從財格 / 從官殺格 / 正格身強 / 正格身弱`
- `dominantFamily`
  - 主導十神家族
- `process`
  - 說明是否跨過 `±210`，以及哪類十神主導

## 範例與校準

### 範例一：P先生
- 四柱：`甲子 / 丙子 / 辛巳 / 乙未`
- 日主：`辛`
- 書上總分：`-197.4`
- 書上結論：`不從`
- 工程重點：
  - 雖然總分很低，但尚未低過 `-210`
  - 因此不能直接判 `從弱`

### 範例二：白韻琴
- 四柱：`甲申 / 壬申 / 戊辰 / 辛酉`
- 日主：`戊`
- 書上總分：`-225.9`
- 書上結論：`從兒格`
- 工程重點：
  - 先跨過 `-210`
  - 再看負分主導家族為 `食傷`
  - 因此判作 `從弱 -> 從兒格`

### 範例三：W先生
- 四柱：`戊戌 / 庚申 / 辛巳 / 辛卯`
- 日主：`辛`
- 書上總分：`+282`
- 書上結論：`從強或專旺`
- 工程重點：
  - 已明顯高於 `+210`
  - 正分主要集中於印比
  - 工程版可先判作 `從強 -> 專旺印比`
