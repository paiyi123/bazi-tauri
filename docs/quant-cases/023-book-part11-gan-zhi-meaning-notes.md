# 第十一篇 干支釋義（254–260）

## 章旨
- 這一篇不是新的算分表，而是在交代：
  - 十天干、十二地支、地支藏干的來源
  - 陰干與陽干在八字系統裡的不同作用
- 對工程最有用的不是新增分數，而是釐清目前模型中哪些差異可以保留為「理論支撐」。

## 書上重點

### 1. 地支藏干的來源
- 作者認為十二地支藏干，對應的是一年十二月裡，夜觀星象時最明顯的主星與旁星。
- 因此：
  - `酉藏辛`
  - `卯藏乙`
  - `子藏癸`
  - `午藏丁己`
  等既有藏干配置，在這一篇得到理論背書。

### 2. 地支主氣與旁氣的觀念
- 各支之主氣，來自「該月最顯明」之五行。
- 旁氣則是同一觀測時段中，同時可見的其他星象五行。
- 這與目前模型用 `0.5 / 0.25 / 0.25` 或 `0.63 / 0.37` 拆主次氣的方式是相容的。

### 3. 陽干略強於陰干
- 作者明說：
  - 陽干與陰干的五行本質能量應視為相等
  - 但陽干因「反射」與「外顯」能力稍強，在八字中的得根、透干表現略勝陰干
- 原文結論可簡化為：
  - `陽干絕不從勢，陰干任從無憑`

## 對目前程式的落地方式

### 已與模型一致的部分
- 地支藏干列表與主次順序：
  - 可沿用目前 `src-tauri/src/quant_model.rs` 的 `hidden_stems_of`
- 地支主氣 / 旁氣拆分：
  - 可繼續由 `hidden_ratios` 與 `score_branch` 依既有比例計分

### 可作為後續微調依據，但本輪不強行數值化
- 陽干得根、透干略強於陰干
- 這一篇沒有給明確係數，因此暫不額外新增乘數
- 目前只把它收錄成理論依據，供後續若要微調：
  - `RootApplication`
  - `TransparencyApplication`
  等權重時參考

## 本篇實作結論
- 第十一篇主要是「理論說明篇」
- 本輪不新增直接改分規則
- 但用來確認：
  - 地支藏干表可繼續沿用
  - 陽干略強於陰干的設計方向有書上根據
