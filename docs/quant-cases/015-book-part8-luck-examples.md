# 第八篇 大運案例節錄（208-218 頁）

這份文件節錄第八篇 `208-218` 頁的案例，作為後續擴充大運／歲運量化的參考。

## 張韶穎成名之運（208-209）

頁碼：

- 書上頁碼：`208-209`

四柱：

- 年柱：`甲子`
- 月柱：`甲戌`
- 日柱：`戊寅`
- 時柱：`丁巳`

原局總分：

- `+57`

當值運程：

- 大運：`辛未運` 中的 `辛運`
- 歲運：`乙酉年`

書上列法：

- 當值大運一字：`辛`
- 歲運一柱：`乙酉`

辛運乙酉年：

- 大運得分：`-23.7`
- 歲運得分：`-24.8`
- 合計：`-48.5`
- `R ≈ 0.65`

書上解讀：

- 雖然是對身強命主有削弱的大運，但因原局夠強，仍屬好運。
- 大運與歲運之沖合，會削弱運值本身，不能只看單一歲運吉凶。
- 原書明言：即使 `乙酉` 與當值 `辛` 入局後發生相沖，整體歲運得分仍不足以扭轉原局偏強的底盤。

- 日柱：`己卯`
- 時柱：`戊辰`

原局總分：

- 約 `+21`

當值運程：

- 大運：`甲申運`
- 歲運：`丙戌年`

書上列法：

- 當值大運一字：`甲申運` 整柱同看
- 歲運一柱：`丙戌`

甲申大運：

- 大運得分：`+1.6`

丙戌歲運：

- 歲運得分：`+47.3`

合計：

- `+48.9`
- `R ≈ 0.92`

書上解讀：

- 對偏強命主而言，這是極強的生旺運。
- 但命主主因未必只是運勢本身，仍要看是否被流年重觸原局問題點。
- 書頁特別示範：`甲申` 大運本身只是小吉，但 `丙戌` 歲運入局後與原局觸發半合／六合／墓庫相沖，才把全年推到 `+48.9`。

## 意外凶死男命出事之運（213-214）

頁碼：

- 書上頁碼：`213-214`

四柱：

- 年柱：`癸巳`
- 月柱：`丙辰`
- 日柱：`甲午`
- 時柱：`乙丑`

原局總分：

- `+5.4`

當值運程：

- 大運：`壬子運` 中的 `子運`
- 歲運：`甲戌年`

書上列法：

- 當值大運一字：`子`
- 歲運一柱：`甲戌`

甲戌年運：

- 大運得分：`+72`
- 歲運得分：`+9`
- 合計：`+81`
- `R ≈ 0.99`

書上解讀：

- 命局總分雖接近中性，但五行流通很差。
- 在高 `R` 的運程衝擊下，容易被放大成重大事件。
- 書上示範了大運與歲運同時入局後的複雜沖合：`子` 與月令 `辰` 先半合，再與日支 `午` 沖，最後又受 `戌` 影響改變化局方向。

## 假從女命 庚戌、戊戌運（217-218）

頁碼：

- 書上頁碼：`217-218`

四柱：

- 年柱：`丁未`
- 月柱：`丙午`
- 日柱：`辛未`
- 時柱：`庚寅`

原局總分：

- `-63`

當值運程：

- 大運一：`庚戌運`
- 大運二：`戊戌運`

書上列法：

- 第一步先看 `庚戌運` 整體對原局的幫扶
- 再看 `戊戌運` 入局後，如何引動 `寅午戌三合化火`

### 庚戌運

- 大運得分：`+60`
- `R ≈ 0.69`

### 戊戌運

- 大運得分：`-90`
- `R ≈ 0.82`

書上解讀：

- 庚戌運為大吉，因為能幫身。
- 戊戌運入局後與寅午形成三合化火，改變原局格局方向，成為大惡運。
- 書上也用這個案例說明：大運確實可能「引動原局質變」，但那是因為入局後改寫了原局合化關係，不是直接改變命局本質強弱。

## 可補記的章節內過渡例（217 頁上半）

這頁在正式進入假從女命之前，先舉了一個「運可否引化原局質變」的過渡例：

- 四柱：`X戌 / 丙申 / 乙X / X午`
- 柱序說明：原書以概念示意，不是完整可驗算案例
- 書上示意：
  - 原局 `戌申` 半合化火，推得約 `-42`
  - 若命局總分約 `-60`
  - 遇 `辛運` 入局後，原局可再與 `丙` 合化水
  - 使整體總分翻到約 `+49.5`

這個例子不是完整命例，但它很重要，因為作者用它鋪陳：

- 大運可以透過入局沖合，讓原局總分大幅改變
- 但作者仍主張「命身強弱的本質不應被直接改寫」
- 因此真正要判的是「大運是否引發原局局部質變」，而不是單純把大運分數線性加到命局上

## 對目前專案的啟發

1. 大運不能只看單純加分，要看 `R` 對原局的相對衝擊。
2. 原局總分接近 `0` 的命局，對運的感受會特別劇烈。
3. 逐年運勢的精算，最後仍需要把：
   - 當值大運一字
   - 歲干
   - 歲支
   - 原局四柱
   放進同一個沖合裁決流程。

## 目前程式第一版落地狀態

目前專案第一版已做到：

- 大運整柱先與原局同場裁決，再抽出 `運干 / 運支` 分數
- 歲運列改為：
  - `原局四柱`
  - `當值大運一字`
  - `流年一柱`
  一起進入同一套沖合裁決

目前較接近書的案例：

- `U 先生未運`
  - 書：`-54`
  - 程式第一版：約 `-52`
- `意外凶死男命 甲戌年`
  - 書：歲運整柱約 `+9`
  - 程式第一版：約 `+9`

目前仍有落差、留待下一輪深化的案例：

- `張韶穎成名之運`
  - 書：`-48.5`
  - 程式第一版仍偏高
- `假從女命 庚戌 / 戊戌運`
  - 書中涉及「大運引動原局質變」
  - 目前第一版尚未完整覆蓋

也就是說，第一版已經把「歲運不只是一柱分數，而會和原局／大運一起裁決」的骨架做進去；但第八篇後半那些更強的質變型案例，仍需要下一輪專修。

## 目前程式第二版落地狀態

目前專案已再往前推一層：

- 歲運列改為：
  - `原局四柱`
  - `大運整柱`
  - `流年一柱`
  一起進入同一個沖合裁決場
- 展示分數仍維持：
  - 前五年看 `運干`
  - 後五年看 `運支`
- 也就是說：
  - `運干 / 運支` 其中未當值的一半，雖不直接列為當值分數
  - 但仍會參與 `三合 / 三會 / 半合 / 六合 / 沖` 的裁決與破局

目前第二版較能反映的例子：

- `張韶穎成名之運`
  - 歲運說明中已可同時看到 `運干辛` 與 `運支酉`
  - 表示 `乙酉年` 不再只是單看 `辛運`，而是整個 `辛酉` 運柱都進場參與裁決
- `意外凶死男命 甲戌年`
  - 歲運說明中已可同時看到 `運干壬` 與 `運支子`
  - 符合書上「子先與辰半合，再與午沖，最後受戌影響」這種需要整個運柱共同參與的讀法
- `U 先生未運`
  - 雖然沒有再疊加單年流年，但第二版的歲運路徑已能把 `己未` 整柱直接送入同一裁決場

目前第二版仍與書有差距的地方：

- `歲運合參` 目前仍是：
  - 當值半運分
  - 加上流年整柱分
- 尚未完全改成書中某些頁面那種：
  - 「整個大運得分」與「整個歲運得分」再做更高層次的合參
- 第八篇後半提到的：
  - 大運引化原局質變
  - 流年再破局
  - 歲運多層優先級反覆覆寫
  這些仍需後續案例驅動再深化

## 目前程式第三版正式回歸重點

目前專案已再往前推到第三版：

- `歲運合參` 可切換：
  - `當值半運合參`
  - `整個大運合參`
- 裁決層已改成：
  - `原局四柱`
  - `大運整柱`
  - `流年一柱`
  一起進入同一個 resolver
- 因此現在已能正式回歸兩類第八篇關鍵歲運現象：
  - 大運先幫身或引局
  - 流年再破局或翻轉

### 假從女命：庚戌運先幫身，癸亥年再破局

四柱：

- 年柱：`丁未`
- 月柱：`丙午`
- 日柱：`辛未`
- 時柱：`庚寅`

原局總分：

- `-63`

大運：

- `庚戌運`

正式回歸的兩個年份：

- `1978 戊午年`
  - 程式第三版：`fullCombinedScore > 0`
  - 說明會直接出現 `三合化火`
  - 代表大運整柱已能引動原局局勢
- `1983 癸亥年`
  - 程式第三版：`fullCombinedScore < 0`
  - `combinedScore < 0`
  - 代表雖然 `庚戌運` 本身仍偏幫身，但 `癸亥年` 入局後足以把當年歲運拉回負面

這組回歸主要鎖的是：

- `庚戌運` 可帶來正向幫扶
- `癸亥年` 可再把該運中的單年推回破局

四柱：

- 年柱：`丁未`
- 月柱：`丙午`
- 日柱：`辛未`
- 時柱：`庚寅`

原局總分：

這組回歸主要鎖的是：

- `戊戌運` 可引動 `寅午戌` 火局
- 第三版下，`整個大運合參` 比 `當值半運合參` 更接近書中「運可引局」的讀法
