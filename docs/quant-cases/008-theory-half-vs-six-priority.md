# 理論案例：半合優先於六合

來源依據：第四篇第 `109`、`116` 頁。

## 書上的結論

書中認為：

- 半合屬於三合家族
- 因此當半合與六合競爭同一支時，半合應優先於六合

最值得落地的讀法是：

- 若 `半合` 與 `六合` 同時想使用同一支
- 且該半合是緊貼半合，或已符合化局條件
- 就不應讓六合先把該支扣弱

## 這在 repo 裡為何重要

如果沒有這條規則，共支命盤會太早先被六合扣分，和書中「力量強者先勝」的方向相反。

## 目前工程化近似

目前 repo 會在下列情況壓掉六合：

- 六合與更強的半合共用同一支
- 且該半合符合以下其一：
  - 緊貼（`distance <= 1`）
  - 已達可化條件

這仍是簡化版，但比把六合與半合視為同一層級更接近書中方向。
