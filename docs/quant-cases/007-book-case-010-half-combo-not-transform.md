# 書本案例 007：申子合而不化

書頁：第 `80` 頁。

## 專案使用命盤
- 年柱：`乙酉`
- 月柱：`甲申`
- 日柱：`丙子`
- 時柱：`乙未`

## 書上讀法

書上明講：

- `地支申子合而不化`
- `因是半合二支分數均無需扣減`

頁面總分為：

- 天干總分：`+130.1`
- 月令：`-60.1`
- 其餘三支：`-47.1`
- 命局總分：`+22.9`

## 這個案例的重要性

這一題直接證明：

- 書上並不是看到半合就必然化

若把 `申子` 強行改成化水，月支 `申` 就會從書中的藏干分數，被壓平為單一化水支，整盤總分會被拉低過頭。

## 目前綁定這題的工程規則

為了貼近書的方向，repo 現在採用：

- 半合只有在下列條件都成立時才可化：
  - 化後五行在可見天干出現，日主也算可見
  - 月令主氣不剋該化後五行

對這盤來說：

- 化後五行會是 `水`
- 可見天干是：`乙 / 甲 / 丙 / 乙`
- 日主是 `丙`
- 盤中沒有可見水干

所以目前 repo 會把 `申子` 保留為：

- `合而不化`

這和書上方向一致。

## 目前 repo 結果

目前量化器輸出已非常接近書頁：

- 天干：`+130.1`
- 地支：`-107.0`
- 命局總分：`+23.1`

和書上的 `+22.9` 幾乎一致，剩餘差距視為四捨五入或版面列法差異。

### 地支逐項對照
- 年支 `酉`：repo `-27.0`，書 `-27`
- 月令 `申`：repo `-60.0`，書 `-60.1`
- 日支 `子`：repo `-41.3`，書 `-41.3`
- 時支 `未`：repo `+21.3`，書 `+21.2`

## 回歸目的

這題應該能防止我們回退到以下錯誤：

- 把所有 `申子` 一律自動化水
- 把 `半合不化` 當成 `六合不化` 一樣乘 `0.52`
