# 第十篇《從格》筆記（235-253）

## 範圍
- 下冊第十篇 `235-253`
- 主題：`從強 / 專旺 / 從弱` 的真假區分、可用神差異、破局與誤判反例

## 章節核心結論
- 第六篇的 `±210` 只夠作第一層臨界判斷。
- 第十篇再往前推一層：
  - `+210 ~ +260`：可視為 `假從強 / 假專旺`
  - `> +260`：可視為 `真從強 / 真專旺`
  - `-210 ~ -260`：可視為 `假從弱`
  - `< -260`：可視為 `真從弱`
- 真從與假從的差別，不只是名字，而會影響：
  - `是否可用食傷`
  - `是否較能承受逆運`
  - `是否容易破局`

## 真從 / 假從工程化判準

### 第一層：從與不從
- `totalScore >= +210`：進入 `從強` 範圍
- `totalScore <= -210`：進入 `從弱` 範圍
- 其餘：`不從`

### 第二層：真從 / 假從
- `totalScore > +260`
  - `真從強 / 真專旺`
- `+210 <= totalScore <= +260`
  - `假從強 / 假專旺`
- `totalScore < -260`
  - `真從弱`
- `-260 <= totalScore <= -210`
  - `假從弱`

### 第三層：子類型
- `從強` 範圍：
  - 正分若以 `印 + 比劫` 顯著佔優，可判 `專旺印比`
  - 否則先判 `從強`
- `從弱` 範圍：
  - `食傷` 主導：`從兒格`
  - `財` 主導：`從財格`
  - `官殺` 主導：`從官殺格`

## 用神規則

### 從強 / 專旺
- 基本原則：
  - 主用 `印比`
  - 忌 `官殺`
  - `財` 亦非首選
- 真從強 / 真專旺：
  - 可兼用 `食傷`
  - 但 `食傷` 不如 `印比`
- 假從強 / 假專旺：
  - 不可直接把 `食傷` 當可用神
  - 更怕逆運破局

### 從弱
- 基本原則：
  - 順其弱勢，用 `食傷 / 財 / 官殺`
  - 忌 `印比`
- 真從弱：
  - 較能承受小逆運
  - 但仍以 `印比運` 最容易破局
- 假從弱：
  - 比真從弱更怕 `印比逆扶`

## 章內理論補充

### 殺印相生不可作為從格可用神
- 書中明確反對：
  - `從強 / 專旺` 用官殺，並藉 `殺印相生` 合理化
  - `從弱` 藉印比相生合理化
- 理由：
  - 殺印相生是一氣生一氣，要求局內氣流通
  - 從格對其逆方五行極敏感，官殺或印比一旦入局，往往就是破局源頭

### 真從較不怕逆運，假從較怕逆運
- 書中用 `+260 / -260` 作為能量更高的第二道線。
- 工程上可把它理解成：
  - 真從：原局主勢能量很大，較不容易被單一步逆運打回正格
  - 假從：只是剛過臨界，較容易在運歲中破局

### 陽干從氣不從勢，陰干從勢無情義
- 書中最終結論是否定把這句當絕對法則。
- 可作為臨界值附近的細緻提示，但不可凌駕量化總分。
- 工程版先不把它硬寫成修正係數，只記為章法說明。

## 範例與校準

### 範例：K先生
- 四柱：`乙卯 / 丙戌 / 丙午 / 戊子`
- 書上總分：`+292`
- 書上結論：
  - `真從強或真專旺`
  - 可用 `木火`
  - 忌 `金水`
- 書中重點：
  - `子午沖` 可成立
  - `卯戌六合` 與 `午戌半合火` 並見時，要比力量
  - 真從強 / 真專旺除 `印比` 外，亦可用 `食傷`

### 範例：W先生之食傷運
- 四柱：`戊戌 / 庚申 / 辛酉 / 辛卯`
- 書上總分：`+282`
- 書上結論：
  - `真從強或真專旺`
  - 可用 `印比`
  - `食傷` 可用，但不如 `印比`

### 範例：男命

- 書上總分：`+269`
- 書上結論：
  - `真從強或真專旺`
  - 行 `食傷 / 金` 運仍能得祿

### 範例：李兆基
- 四柱：`己巳 / 丁卯 / 甲寅 / 戊辰`

- 書上結論：
  - `從強或專旺`
  - 可用 `水木`
  - 忌 `土金`
- 工程解讀：
  - 此分數尚未越過 `+260`
  - 可作 `假從強 / 假專旺` 的校準例
