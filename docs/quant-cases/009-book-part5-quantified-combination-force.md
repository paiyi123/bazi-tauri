# 書本筆記：第五篇 合化力量量化

來源範圍：第五篇，第 `123-140` 頁。

## 為何整理這份筆記

第四篇已經告訴我們：

- `三會 > 三合 > 半合 > 六合`
- `力量為王`

但第四篇主要還是在講原理與優先級，沒有把各種合局的力量數值真正量化。第五篇才補上這一塊。

這份筆記的目的，就是把第五篇裡和程式最有關的數值整理出來，作為未來建立「誰勝誰負」數值判斷層的依據。

## 穩定結論

### 1. 六合不化與六合化，是兩種不同模型

第 127-129 頁指出：

- `六合不化`
  - 雙方會減弱
  - 書中的剩餘力量大約是 `0.52`
- `六合化`
  - 已不是單純兩支互相耗損
  - 化後總力量大約落在 `1.65 ~ 1.80`
  - 工程上可先用平均值 `1.72`

這表示：

- `六合不化` 屬於「餘力模型」
- `六合化` 則屬於「化局力量模型」

### 2. 半合化強於六合化

第 129-130 頁量化了半合化：

- 緊貼半合：`2.05`
- 隔一位：`1.875`
- 隔二位：`1.625`

這表示：

- `半合化` 的力量高於 `六合化`
- 半合化的力量還會受到距離影響

### 3. 半合不化仍然保有專氣凝聚

第 131、139、140 頁指出：

- `半合不化` 不是像 `六合不化` 那樣單純耗損
- 緊貼的 `半合不化` 仍帶有約 `1.25` 的專氣力量

這表示：

- `半合不化` 不能直接等同 `0.52`
- 即使不化，也會提升相關五行的氣勢

### 4. 三合化屬完整高階合局

第 131-132 頁量化了 `三合化`：

- 總力量：`3.44`
- 分配為：
  - 專氣支：`1.40`
  - 其餘兩支：各 `1.02`

這表示：

- `三合化` 明顯強於 `半合化`
- 三支在局中的角色不同，力量分配也不同

### 5. 三合不化仍有明顯力量

第 133 頁指出：

- `三合不化` 的總力量為 `1.89`

這表示：

- 即使沒有完全化成，完整三合局仍比 `半合不化` 更有力

### 6. 三會化是最強的地支凝聚型合局

第 132-134 頁量化了 `三會化`：

- 總力量：`5.25`
- 分配為：
  - 專氣支：`2.10`
  - 長生支：`2.10`
  - 墓庫支：`1.05`

這也從數值上支持了第四篇的結論：

- `三會 > 三合`

### 7. 三會不化也仍然很強

第 133-134 頁指出：

- `三會不化` 的總力量為 `3.41`

這表示：

- 就算不化，三會仍遠高於半合與六合

## 力量表摘要

| 類型 | 合化 | 不化 |
|---|---:|---:|
| 六合 | `1.65 ~ 1.80`（平均約 `1.72`） | `0.52` 餘力模型 |
| 半合 | 緊貼 `2.05`；隔一位 `1.875`；隔二位 `1.625` | 緊貼約 `1.25` |
| 三合 | `3.44` | `1.89` |
| 三會 | `5.25` | `3.41` |

## 這份筆記解答了什麼

這一篇很清楚回答了我們之前卡住的問題：

- `三合` 與 `半合` **不是同一組參數**
- `六合 / 半合 / 三合 / 三會` 各自都有不同的合化與不化力量模型

也就是說，未來程式若要更像書，至少要把兩件事分開：

1. 地支逐列計分
2. 沖合誰勝誰負的力量比較

目前 repo 在第 `1` 點已經比以前成熟很多，但第 `2` 點還只做到部分。

## 建議的工程方向

最穩的下一步應該是：

1. 先維持現有書本案例的列分穩定
2. 另外建立一層 `六合 / 半合 / 三合 / 三會` 的局力比較模型
3. 再用第五篇的數值決定：
   - 誰先成立
   - 誰會被沖打散
   - 共支時誰優先
   - 是否還有二次合化的餘力

## 已知限制

- 這些總力量數值，不能直接一比一套進目前 UI 的 `本分 / 位調 / 增值`
- 書中仍然強調時序與語境，不能只看一個乘數
- App 量化模型目前尚未完整實作第五篇的局力比較層
