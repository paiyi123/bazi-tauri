# 第三篇《用神》84-97 頁筆記

這份筆記整理上冊第三篇 `84-97` 頁，重點是把書中的「用神判斷」轉成可工程化的第一版規則。

## 核心方法

書中的主線很清楚：

1. 先用量化五行力判定命局 `身強 / 身弱`
2. 再看局中是否 `見印`
3. 依印星出現在 `天干 / 地支 / 全無`，決定基本用神
4. 最後回頭檢查局中是否有局部偏患，避免把會助旺忌神的五行誤列為可用

作者在 `85-87` 頁反覆強調：

- `調候法`、`三審法`、`病藥法` 都只能作輔助參考
- 不能跳過強弱判斷，直接靠某一套傳統口訣決定用神
- 最佳做法是：
  - `先判命身強弱`
  - `初步定出用神`
  - `再加以調候病藥等考慮`
  - `對局後才決出真正用神`

## 局部偏患修正

`87` 頁之後作者補了一個很重要的限制：

- 即使量化總分已判成身強或身弱
- 也不能忽略局中局部地區的偏患
- 某些原本可用的五行，可能因為會再去生助局部忌神，而必須轉成 `慎用`

書中的代表性說法：

- 乙日身弱，時干見壬印
- 按基本規則應可用 `官印比`
- 但若局中 `寅午半合化火`，又透 `丙` 於月干
- 火氣過旺時，`木` 會再生火
- 則木不宜再列為正式用神，只可用 `官印`

這一步對應到程式裡的第一版實作：

- 先依 `身強 / 身弱 × 見印位置` 產生基本用神
- 再抓出局中最突出的偏患五行
- 若某個基本用神會直接生助該偏患，則把它從「可用」移到「慎用」

## 範例：L 女士

頁面：`88-90`

四柱：

- 年柱：`己未`
- 月柱：`壬申`
- 日柱：`乙卯`
- 時柱：`壬午`

重要數值：

- 天干總分：`+54`
- 月令申：`-22.5`
- 年支未：`-14.1`
- 日支卯：`+55`
- 時支午：`-27.5`
- 地支總分：`+13.4`
- 命局總分：`+45`

書上結論：

- `乙日元在月令並未失令，自然是身強`
- `可用火土`
- `忌水木`
- `由於天干二印，亦忌金`

對應規則：

- `身強`
- `天干見印`
- 因此按基本表：
  - 用：`食財` -> `火土`
  - 忌：`官印比` -> `金水木`

這是目前最適合拿來鎖第一版回歸的正式案例。

## 範例：S 先生

頁面：`94`

四柱：

- 年柱：`癸未`
- 月柱：`壬戌`
- 日柱：`乙丑`
- 時柱：`丙子`

頁面上的單柱數值：

- 年柱：`癸未 = +41 / -8`
- 月柱：`壬戌 = +53 / -64`
- 日柱：`乙丑 = 日元 / -4`
- 時柱：`丙子 = -25 / +31`

作者重點說明：

- 此局 `乙日元命主身強`
- 局中有四個用神：`丙、子、未、戌`
- 丙火坐子水受剋，發揮有限
- 三土用神較為得令，其中力量最大的是 `戌`

這個案例告訴我們：

- 第三篇真正完整的用神判斷，不只會停在「可用火土、忌水木金」
- 還會繼續分辨：
  - 哪些柱位是具體用神
  - 哪一個用神力量最大
  - 是否彼此能 `通關相生`

目前程式尚未做到這一層，先列為後續校準案例。

## 範例：M 先生

頁面：`95`

四柱：

- 年柱：`庚申`
- 月柱：`壬午`
- 日柱：`乙丑`
- 時柱：`壬午`

頁面上的單柱數值：

- 年柱：`庚申 = +53 / -18`
- 月柱：`壬午 = +46 / -25`
- 日柱：`乙丑 = 日元 / -10`
- 時柱：`壬午 = +52 / -51`

作者重點說明：

- `M 先生是身強`
- 四個負分地支皆為用神
- 但 `二午受剋失令`
- 申支力量有限
- 真正可算得令的，是 `時干壬正印` 與 `食神庚`

這再次說明：

- 第三篇的深層版本，不只決定五行與十神
- 還會進一步評估：
  - 哪個用神 `得令`
  - 哪個只是 `存在但力量有限`

## 支運為用也需小心

頁面：`96`

這頁是章內的轉折提醒。

作者舉一個簡化盤說明：

- 命局可用 `火`
- 但若走 `未運`
- 未中的 `乙` 透於干，生日主之力勝過洩耗之能
- 雖然未的主氣看似為吉
- 實際上未運得分只有 `+9.2`

重點不是四柱本身，而是：

- `支運可用` 不代表整步大運必吉
- 必須把 `支運透出情況` 也算進去

這一頁和第八篇大運系統可以直接互相呼應。

## 範例：許瑋倫

頁面：`97`

四柱：

- 年柱：`戊午`
- 月柱：`癸亥`
- 日柱：`己卯`
- 時柱：`戊辰`

大運：

- `80 壬戌`
- `90 辛酉`
- `00 庚申`

作者重點說明：

- 己日元雖失月令，但得勢得地，仍屬 `身強`
- 原以為申運是洩耗身強日主之吉運
- 但 `原局年月干戊癸合而不化`，二干分數 `×0.52`
- `申` 中戊氣透出在戊癸二干，其增值也要再乘 `0.52`
- 因此申運雖看似吉，實際只有小吉，且逢流年更容易被引發事故

這頁對程式的意義：

- 用神判斷不能只看原局
- 走運時，用神是否仍成立，還要看：
  - 運柱入局後的天干合化
  - 支氣透出後是否放大原局隱患
