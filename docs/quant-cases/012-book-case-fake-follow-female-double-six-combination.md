# 案例：假從女命，`未午未` 夾合壓過 `寅午` 半合

## 命盤
- 年柱：`丁未`
- 月柱：`丙午`
- 日柱：`辛未`
- 時柱：`庚寅`
- 日主：`辛`

## 書上重點
- 表面同時存在：
  - `寅午` 隔一位半合
  - `未午` 兩組六合
- 但此局不是單一六合，而是 `未午未` 的夾合。
- 書中以第五篇精算法比較局力：
  - 單一 `午未` 六合總能量：`0.96`
  - `未午未` 兩組六合總能量：`1.92`
  - `寅午` 隔一位半合總能量：`1.88`
- 因為 `1.92 > 1.88`，所以：
  - `未午未` 夾合壓過 `寅午` 隔位半合
  - `寅午半合化火` 不成立
  - 月令 `午` 被二未合住，力量只剩原本分數的 `0.04`

## 命理結論
- 此局不是從殺
- 而是正格身弱
- 可用：`土、金`
- 忌：`水、木、火`
- 書上總分：約 `-63`

## 目前程式回歸
目前程式已補進：
- `未午未` 夾合力量疊加
- 疊加六合壓過 `寅午` 隔位半合
- 被夾住的 `午` 以 `0.04` 餘力處理

目前回歸結果：
- 天干總分：`-39.0`
- 地支總分：`-26.1`
- 總分：`-65.1`
- 判定：`偏弱`

## 差距說明
- 與書上的 `約 -63` 已相當接近
- 剩餘差距主要來自：
  - 書頁是「約數」敘述，不是完整逐列總表
  - 程式仍用工程化分數表去計算二未本身與寅支的細項

## 對應規則
- `未午未` 夾合屬於「同一支同時受兩個同性六合牽制」
- 目前實作做法：
  - 若某支同時被左右兩組六合夾住，且兩組六合目標五行相同，視為疊加夾合
  - 疊加六合總能量若大於競爭半合總能量，則壓掉該半合
  - 被夾住的中心支以 `0.04` 餘力計分
