# 第八篇 大運（196-207 頁）筆記

這份筆記整理《量化五行力（下）》第八篇 `196-207` 頁可直接落地的規則。

## 章節主軸

- 大運一字管五年，歲運管一年。
- 一年運程不是單純把運分數加到原局，而是看運值對原局的相對衝擊。
- 大運不能改變命局本質強弱，但會透過沖合與入局互動，大幅改變原局總得分。

## 可直接工程化的規則

### 1. 大運基本值

- 大運基本分值：`60`
- 歲運基本分值：`40`
- 干支各佔：`20`

目前專案只有大運列表，尚未做到逐年歲運推算，因此先把第八篇真正能落地的一層補到「大運相對影響」。

### 2. 大運相對影響 R

書中定義：

- 命局總分：`M`
- 大運得分：`L`
- 大運對命局影響：`R = L / sqrt(M^2 + L^2)`

作用後命局：

- 當命局為正分強身：`M' = M * (1 + R)`
- 當命局為負分弱身：`M' = M * (1 - R)`

可以統一寫成：

- `M' = M * (1 + R * sign(M))`

這一層已接進目前 `src-tauri/src/quant_model.rs` 的大運流年計算，並輸出：

- `R值`
- `作用後命局`

### 3. 大運入局判則

書中列了幾個很重要的原則：

1. 大運入局後，與原局四柱之間的沖、合、化，都按原局同類規則判。
2. 大運干可以得根或透干於原局。
3. 大運干支是依次入局，不是同時完整入局。
4. 歲運干支同時入局。
5. 歲運干可以根於歲運支，也可以根於當值大運支。
6. 大運入局後，會影響原局合化是否成立。
7. 大運被合時：
   - 天干五合／地支六合不化：得分乘 `0.52`
   - 三合／三會／半合不化：相關專氣仍會提升
8. 大運受沖時，要按沖量化處理。
9. 大運入局的沖合變化，仍依第四、五篇的優先順序判。
10. 干運有可能形成第二次合化，但條件嚴格。
11. 干運若與日元相沖或相合，干運本身得分不減，但其對命主的影響仍存在。
