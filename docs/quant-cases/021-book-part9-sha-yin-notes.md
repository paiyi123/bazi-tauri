# 第九篇《殺印相生》筆記（219-234）

這份筆記直接依下冊第九篇書頁圖片整理，不走 OCR。

主題是：當命局裡的 `殺` 並非單純剋日元，而是經由 `印` 轉正時，原本的負能量可以改判成正能量。

## 一、核心規則

### 1. 天干殺印相生

第九篇最明確、也最常見的鏈條是：

- 年干：`官 / 殺`
- 月干：`印`
- 月干生日元

書上的語意是：

- `年干七殺生月干印`
- `月干印生日元`

工程化第一版可整理成：

1. 年干十神必須屬 `官 / 殺`
2. 月干十神必須屬 `印`
3. 年干五行要生月干五行
4. 月干五行要生日元五行
5. 年干原始得分必須是負分
6. 月干（以及其他天干印星的助力）必須足以承受這個負分

在分數處理上，第九篇的寫法是：

- 若年干殺分為 `-53.2`
- 則可把它翻成 `+53.2`
- 等於總分要加回 `53.2 × 2`

所以可用公式：

- `轉正後參考總分 = 原局總分 + |殺分| × 2`

### 2. 地支殺印相生

書上只明確承認「純氣鏈」的兩種情況：

1. `酉 -> 子 -> 木日元`
   - 年支酉生月支子
   - 月支子生日元甲 / 乙

2. `子 -> 卯 -> 火日元`
   - 年支子生月支卯
   - 月支卯生日元丙 / 丁

也就是說，地支要成立殺印相生，不能是一般雜氣支亂接，而要是：

- 年支為專氣之 `酉` 或 `子`
- 月支為專氣之 `子` 或 `卯`
- 月支再去生日元

### 3. 土不視為可支持命局內殺印相生的專氣

第 234 頁明講：

- 土並非專氣
- 不支持命局中的殺印相生

所以像：

- `丙 -> 己 -> 庚`

這種看起來像火生土、土生金的鏈條，在命局內不能直接判成殺印相生。

## 二、成立條件的細化

### 1. 不是只看鏈條成立，還要看印能否承殺

小沈陽一例非常關鍵：

- 天干確實有 `辛 -> 癸 -> 乙`
- 但辛殺力量 `-64.2`
- 癸印力量 `+57.2`

因為：

- `64.2 > 57.2`

所以：

- `印不敵強殺`
- `負能轉正不成立`

因此第一版實作採用：

- `總印分 >= |來源殺分|` 才判為可轉正

這裡的「總印分」不只看月干印，也納入其他天干印星對月印的支援。

### 2. 不直接改原量化總分，只另給「轉正後參考總分」

第九篇的章法和主量化模型不同，屬於另一條「負能轉正」邏輯。

為避免直接改寫主量化總分、影響已建立的大量回歸，這一輪實作採：

- 保留原局 `quantModel.totalScore`
- 額外輸出 `殺印相生分析`
- 若成立，給出：
  - 來源殺分
  - 印支援分
  - 轉正後參考總分

## 三、案例

### 案例 M 先生（224-225）

四柱：

- 年柱：`庚午`
- 月柱：`壬午`
- 日柱：`乙丑`
- 時柱：`壬子`

書上原局總分：

- `-58.2`

書上關鍵天干分：

- 年干庚：`-53.2`
- 月干壬：`+52.8`
- 時干壬：`+46`

書上結論：

- 庚年干負能可轉正
- `-58.2 + (53.2 × 2) = +48.2`

也就是：

- 天干殺印相生成立
- 命局由弱轉強

### 案例 C 先生（226-229）

四柱：

- 年柱：`庚戌`
- 月柱：`壬午`
- 日柱：`甲申`
- 時柱：`丁卯`

書上原局總分：

- `-96.5`

書上關鍵天干分：

- 年干庚：`-44.7`
- 月干壬：`+48.4`

書上結論：

- 庚年干負能可轉正
- `-96.5 + (44.7 × 2) ≈ -7`

這題很關鍵，因為作者進一步說明：

- 如果把殺印相生的負能轉正算進來
- 整體就變成接近五行流通之局

### 案例：小沈陽（230-231）

四柱：

- 年柱：`辛酉`
- 月柱：`癸巳`
- 日柱：`乙酉`
- 時柱：`丁亥`

書上關鍵天干分：

- 年干辛：`-64.2`
- 月干癸：`+57.2`

書上結論：

- 雖然外形是 `辛 -> 癸 -> 乙`
- 但 `辛殺 > 癸印`
- 印不敵強殺
- 負能轉正不成立

### 理論驗證例（232）

四柱：

- 年柱：`丙子`
- 月柱：`辛卯`
- 日柱：`丙子`
- 時柱：`癸巳`

書上說明：

- 原局總分約 `-20`
- 若年支子殺能透過月支卯印轉正
- 總分可由 `-20` 變成 `+20`

這題屬理論驗證，書上沒有完整逐列表，所以目前先收錄為章法例，不作精確回歸。

## 四、工程化第一版

目前程式已落地：

1. `天干殺印相生`
   - 檢查 `年干殺 -> 月干印 -> 日元`
   - 以「總印分是否大於等於來源殺分」決定能否轉正

2. `地支殺印相生`
   - 只接受：
     - `酉 -> 子 -> 木日元`
     - `子 -> 卯 -> 火日元`

3. `轉正後參考總分`
   - `原局總分 + |來源殺分| × 2`

## 五、目前限制

這一版尚未做：

- 地支雜氣支下更細的殺印相生分攤
- 把「殺印相生轉正」直接回寫進主量化總分
- 大運 / 歲運入局後的殺印相生轉正

但章內最明確的三個主案例：

- `M 先生`：成立
- `C 先生`：成立
- `小沈陽`：不成立

都已可作為正式測試分支。
