# 第四篇後段 Pattern 範例（119-122 頁）

這份文件整理第四篇後段沒有完整四柱分數表、但可直接轉成程式規則的 pattern 範例。

## 1. 二次合化的基本條件

來源頁：`119-120`

### 書上重點

- 第一個合化局先成立後，才看是否還有餘力支撐第二次合化
- 只有合化前後性質幾乎不變者，才最可能第二次合化
- 四專氣支 `子、午、卯、酉` 最典型
- 六合中又以 `酉` 最具代表性

### 工程規則

- `原支五行 == 化後五行`
- 且原支屬 `子/午/卯/酉`
- 可標記為「可再合化候選」

### 目前實作狀態

- 已補成 `candidate eligibility`
- 尚未把第二次合化的數值分配全面寫進 score engine

## 2. 左右六合局並見：化神距離優先

來源頁：`120-121`

### Pattern

- `子 / 卯 / 戌 / 卯`
- 日主用火，左右兩側 `卯` 都足以與中間 `戌` 成 `卯戌六合化火`

### 書上判準

- 若兩側都能成立，不先比總力量
- 先看化神靠近哪一側
- 若距離相同，再比位置分值

### Repo 測試對應

- `leftRightSixCombinationPrefersHigherWeightSideWhenTransformStemDistanceIsEqualForMonthAndHour`
- 期望：
  - `月支卯` 與 `日支戌` 成立
  - `時支卯` 不成立

## 3. 左右六合局並見：位置分值決勝

來源頁：`121`

### Pattern

- `卯 / 戌 / 卯 / 子`
- 化神在中間位置時，左右候選到化神距離相同

### 書上判準

- 距離相同時，改比位置分值
- `日支 55 > 年支 45`
- 所以應取 `日支卯`，不是 `年支卯`

### Repo 測試對應

- `leftRightSixCombinationPrefersHigherWeightSideWhenTransformStemDistanceIsEqualForYearAndDay`
- 期望：
  - `月支戌` 與 `日支卯` 成立
  - `年支卯` 不成立

## 4. 化學變化

來源頁：`121-122`

### 書上重點

- 物理力量決定「哪個合局真正成立」
- 但配偶宮、感情傾向、人際關係等，可能同時保留左右兩側的化學影響

### 工程策略

- 先不進分數
- 後續若要做，較適合做成 `解釋層`，不直接改 `score`
