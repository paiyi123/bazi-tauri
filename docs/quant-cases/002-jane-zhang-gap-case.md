# 量化差距案例 002：Jane Zhang 公開生日近似盤

## 為何保留這個案例

這個案例用來驗證目前專案的量化模型，和《量化五行力》風格案例之間的差距。

它會被刻意保留，即使目前實作還**沒有**追上書上的判斷。

## 專案內使用的輸入
- 陽曆：`1984-10-11 07:00:00`
- 性別：女
- 備註：使用公開生日近似盤，重點是對齊坊間流傳的張靚穎命盤形狀

## 命盤
- 八字：`甲子 甲戌 戊寅 丙辰`
- 日主：`戊`
- 前三步大運：`癸酉`、`壬申`、`辛未`

## 目前 App 量化結果
- 摘要：量化五行力總分 `-20.5`，判定為偏弱
- 判定：`偏弱`
- 天干總分：`-54.5`
- 地支總分：`34.0`
- 命局總分：`-20.5`

## 書本側的預期
- 書中描述結果：約 `+57`
- 判定：`身強`
- 書中特別強調的原因：
  - 戊日元在時支得祿，可逆轉地支形勢
  - 二甲在地支得祿根，力量很大
  - 透干與得根的放大效果，會明顯改變整體判定

## 為何目前專案仍有差距

目前量化器在這幾塊仍不完整：

1. 非日主天干的得根放大仍偏保守
2. 祿根修正還不夠強，無法完整重現書中的翻盤力道
3. 透干增值還沒完全做到書中的層次
4. 複雜地支沖合的先後順序，仍屬 MVP 級

## 這個案例的價值
- 適合做未來 `祿根 / 透干 / 高階修正` 的目標案例
- 能持續提醒專案：目前雖然基礎計分已接近書，但進階修正仍未完成
