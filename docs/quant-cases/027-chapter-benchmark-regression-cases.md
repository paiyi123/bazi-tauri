# 章節基準回歸案例整理

這份文件整理《量化五行力》目前最值得保留為「章節基準回歸」的案例。

目的不是把案例硬寫成特例，而是把書中的代表盤當成校準護欄：

- 後續補新規則時，可立刻看出是否把舊章節的數值拉壞
- 可分辨問題出在：
  - 局勢裁決
  - 合化量化
  - 沖後殘量
  - 天干得根 / 透干二次放大
- 可避免模型在某一章修正後，把另一章的代表盤炸高或壓低

## 保留原則

適合保留為章節基準回歸的案例，通常至少符合一項：

- 定義某一章最核心的規則邊界
- 很容易因後續補規則而漂移
- 能明確暴露重複計算或局勢優先級錯誤
- 書上已有四柱、重要柱分、總分與明確結論

不一定每個案例都要鎖到小數點，但至少應鎖：

- 四柱
- 關鍵 `combineNote`
- 關鍵柱分
- 總分容差
- 格局 / 從格 / 用神結論

## 建議保留的章節基準回歸

### 第四篇 沖合論

- 案例五：`戊申 / 戊午 / 丙寅 / 己亥`
  - 作用：`寅午半合化火` 基準盤
- 案例九：`壬辰 / 乙巳 / 丙子 / 辛卯`
  - 作用：`子辰半合化水`
- 案例十：`乙酉 / 甲申 / 丙子 / 乙未`
  - 作用：`申子合而不化`
- 左右合局並見例
  - 作用：比對 `半合 / 六合 / 左右合局並見` 的優先級

### 第五篇 精算五行力

- 白韻琴：`甲申 / 壬申 / 戊辰 / 辛酉`
  - 作用：`辰酉六合化金 = 1.0 金 + 0.8 金 + 0.2 餘氣`
- 特殊命例：`癸巳 / 甲子 / 己未 / 庚午`
  - 作用：`巳午未三會不化` 與 `甲庚隔一位沖 ×0.875`
- 李兆基：`己巳 / 丁卯 / 甲寅 / 戊辰`
  - 作用：`寅卯辰三會化木`
- K先生：`乙卯 / 丙戌 / 丙午 / 戊子`
  - 作用：`卯戌六合化火 + 子午沖化零`
  - 特別重要：最能抓「地支列分、透干、得根重複取氣」
- W先生：`戊戌 / 庚申 / 辛酉 / 辛卯`
  - 作用：`申酉戌三會化金`
  - 特別重要：最能抓三會後對 `庚辛` 的過度放大

### 第六篇 從與不從

- `甲子 / 丙子 / 辛巳 / 乙未`
  - 作用：明確 `不從`
- `甲申 / 壬申 / 戊辰 / 辛酉`
  - 作用：`從弱 -> 從兒格`
- `戊戌 / 庚申 / 辛巳 / 辛卯`
  - 作用：`從強 -> 專旺印比`

### 第七篇 精算案例

- `丙寅 / 丙申 / 戊寅 / 甲寅`
  - 作用：`二寅夾沖申`
- `丁亥 / 庚戌 / 己巳 / 庚午`
  - 作用：`戌午雙半合共享專氣支`
- `庚戌 / 丁亥 / 壬子 / 丙午`
  - 作用：`子午沖封丙丁祿根`

### 第八篇 大運

這一篇實務上應保留的 benchmark，不只包含單一步大運，也包含：

- 大運入局
- 流年入局
- 歲運與原局共同裁決

建議保留：

- `庚戌運先幫身，癸亥年再破局`
  - 作用：同一大運下，流年可把方向重新拉回負面
- `戊戌運引動寅午戌三合化火`
  - 作用：運可引動原局質變
- 一個「前五年看運干、後五年看運支」基準例
  - 作用：歲數與半運模式護欄

### 第九篇 殺印相生

### 第十篇 從格進階

- K先生：`乙卯 / 丙戌 / 丙午 / 戊子`

  - 作用：真專旺上限校準

## 不必都做成完整總分回歸的項目

以下較適合保留為規則單測，而不一定要做完整總分 benchmark：

- 單一倍率：
  - `0.52`
  - `0.75`
  - `0.875`
- 單一 pair / triplet 是否成立
- 單純 `combineNote` 文案
- 單純 `可再合化候選`

## 建議實作策略

1. 每章保留少量 benchmark，不求案例越多越好。
2. 章節 benchmark 不做特判，而是當成回歸護欄。
3. 若新規則讓 benchmark 大幅漂移，先找重複取氣或優先級錯判，不要先加 cap。
4. 對仍未完全對上的章節案例，可先標成：
   - `pending benchmark`
   - `expected gap`
   再逐步收斂。
