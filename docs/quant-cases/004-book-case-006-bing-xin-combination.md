# 書本案例 004：案例六 丙辛合

## 命盤
- 八字：`丙寅 辛丑 壬戌 庚戌`
- 日主：`壬`

## 書頁重點
- 核心判斷：`丙辛合水，但合化不能成立`
- 書頁使用 `0.52` 作為丙辛合後的剩餘力量比例
- 丙、辛兩干的位調分都要先乘 `0.52`
- 凡支氣透到丙、辛者，其增值也要再乘 `0.52`
- 書頁總分為：`+33`

## 目前程式狀態
- 天干五合的剩餘力量規則已實作
- 包含：
  - `甲己`
  - `乙庚`
  - `丙辛`
  - `丁壬`
  - `戊癸`
- 以本案例來看，目前程式已能把總分與各列數值拉到和書接近的區間

## 這個案例的重要性
- 驗證天干五合不只是說明文字，而是真的會改變計分
- 驗證透干增值也會被五合剩餘力量一起壓低
- 是後續繼續補「合化成立 / 合而不化」邊界時的主回歸案例
