# 第七篇《精算算命顯神通》案例筆記（165-195）

## 範圍
- 下冊第七篇 `165-195` 頁
- 主題：把第五篇的精算五行力與第四篇沖合論，直接落到完整命例

## 本篇已落地規則
- `寅申近沖`：長生支受沖支氣化零，只留未受沖主氣
- `巳亥沖`：可進一步細拆成 `丙壬沖`、`甲庚沖`
- `巳午未三會不化`：火專氣提升至 `1.45 / 1.45 / 0.5`
- `未午未` 雙六合夾住月令：中間支只剩 `0.04`
- `申子辰` 雙三合不化：專氣支提升到 `1.3`，支援支提升到 `0.5`
- `子午沖`：二支列分歸零，並封鎖丙丁於午取祿根
- `戌午戌` 共享專氣支雙半合：午提升到 `1.3`，兩戌分別為 `0.8 / 0.7`，餘氣保留 `0.2 / 0.3`

## 仍屬已知 gap 的規則
- `寅亥六合` 緩和 `巳亥沖` 的 `0.64 / 0.308` 緩衝公式
  - 目前已有候選與部分沖後剪枝
  - 但書上那種「六合力抵消沖力後，支氣只損失 36%」的完整分攤，尚未完全通用化

- 關鍵規則：

- 程式狀態：已正式回歸

### 案例五：楊啟彥
